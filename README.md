# SpeedTest 14115 ⚡

Test de velocidad de internet con interfaz cyberpunk/neón. Mide descarga y subida usando servidores de Cloudflare.

## Características

- **Dos modos de prueba**: Rápida (~12s, 3 rondas) y Detallada (6 rondas)
- **Múltiples unidades**: Mbps, MBps, kbps, kBps, Gbps, GBps
- **Métricas avanzadas**: Percentil 90, velocidad media, mínima y desviación estándar
- **Gráfico en tiempo real** de velocidad y histograma de distribución
- **Diagnóstico avanzado** con análisis de consistencia y estabilidad del enlace
- **Diseño responsive** (landscape y portrait)
- **APK disponible** para Android

## Cómo usar

1. Abre `index.html` en cualquier navegador moderno
2. Selecciona el modo (Rápida o Detallada)
3. Pulsa **INICIAR**
4. Espera los resultados

No requiere servidor ni instalación. Funciona offline (excepto la prueba de velocidad que necesita conexión).

## Tecnologías

- HTML5 / CSS3 / JavaScript vanilla
- Fuentes: Orbitron, Rajdhani, Share Tech Mono (Google Fonts)
- Servidor de prueba: `speed.cloudflare.com` (archivos de 25MB)
- Sin dependencias externas ni frameworks
